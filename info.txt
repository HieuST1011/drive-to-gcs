Sync Drive to GCS
- Hệ thống kết nối Google Drive
  - Hệ thống 1 user (dành cho Sync Drive)
  - OAuth App
    - Danh sách các quyền
  - OAuth Login Google Drive
  - Cấu hình thư mục cần đồng bộ
  - Cấu hình Drive Push Notification
    - https://developers.google.com/drive/api/reference/rest/v3/files/watch?hl=vi&authuser=0
    - https://developers.google.com/drive/api/reference/rest/v3/channels?authuser=0&hl=vi#Channel
  - Các cấu hình được lưu vào database
  - Xóa cấu hình Drive Push Notification
    - https://developers.google.com/drive/api/reference/rest/v3/channels/stop?hl=vi&authuser=0
- Webhook nhận Drive Push Notification khi có changes on file
  - Push message vào Kafka
  - Handle message trong kafka
    - Refresh OAuth Token
    - Truy cập file có changes
    - Upload file lên GCS
